URL Shortener Microservice

Pass a URL as a parameter and receive a shortened URL in the JSON response.

Example Input:

https://amusing-harmonica.glitch.me/new/http://www.example.com/
https://amusing-harmonica.glitch.me/new/http://foo.com:80

Example Output:

{ "original_url":"http://www.example.com/", "short_url":"https://amusing-harmonica.glitch.me/123" }
